# VibeFlow – AI Resume Analyzer

## Overview

VibeFlow is an AI-based system that analyzes resumes using Natural Language Processing and Machine Learning techniques. It evaluates resume quality and provides improvement suggestions.

## Features

* Resume parsing and skill extraction
* Resume scoring system
* NLP-based analysis
* User-friendly interface
* Secure local processing

## Technologies Used

* Python
* Machine Learning
* Natural Language Processing (NLP)
* Flask
* HTML / CSS / JavaScript

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py
